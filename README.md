# üìÖ Calendar  - React App

A **production-ready Google Calendar clone** built with React. This app features a monthly calendar view, event display, conflict detection, and a responsive, clean UI. Perfect for learning, customization, or production use.

---

##  Key Features

### Core Functionality
-  Monthly calendar view with full grid layout
-  Navigation controls (Previous/Next month, Today button)
-  Current date highlighting with visual indicator
-  Event display from static JSON data
-  Conflict detection ‚Äì overlapping events are flagged
-  Responsive design with clean UI/UX

### Advanced Features
- Smart event organization: Events grouped by date
- Conflict detection algorithm using time interval overlap
- Color-coded events for holidays vs regular events
- All-day event support
- Smooth hover effects and interactive feedback
- Today indicator for the current date
  ##  Setup Instructions

### Prerequisites
- Node.js (v16 or higher)
- npm (comes with Node.js)
- Code editor (VS Code recommended)

npx create-react-app calendar-app
